# 学科标注标签规范

## 背景

用户反馈：生成的图片上标注内容有英语语法标签（主语、谓语、宾语），但书是数学书。需要根据学科定制标注内容。

## 学科标签映射

### 英语
- 边栏笔记：主语、谓语、宾语、状语、表语、定语
- 圈出的术语：语法术语（如 that, which, who, where）
- 批注风格：语法分析型

### 数学
- 边栏笔记：公式、定理、解题关键、易错、步骤1、步骤2
- 圈出的术语：数学符号和公式
- 批注风格：解题思路型

### 语文
- 边栏笔记：修辞手法、中心思想、关键词、段落大意、写作手法
- 圈出的术语：修辞手法名称
- 批注风格：文学分析型

### 文言
- 边栏笔记：实词、虚词、古今异义、词类活用、特殊句式
- 圈出的术语：文言词汇
- 批注风格：古文解析型

### 物理
- 边栏笔记：公式、原理、受力分析、解题关键、易错点
- 圈出的术语：物理量和单位
- 批注风格：原理推导型

### 化学
- 边栏笔记：反应方程式、实验步骤、注意安全、易错
- 圈出的术语：化学物质和反应
- 批注风格：实验操作型

### 通用（默认）
- 边栏笔记：重点、难点、考点、易错、必背、理解
- 圈出的术语：关键词
- 批注风格：通用学习型

## 实现方式

在 `aging_engine.py` 的 `generate_page_prompt()` 函数中：

```python
subject_labels = {
    '英语': 'grammar labels like 主语, 谓语, 宾语, 状语, 表语, 定语',
    '数学': 'math labels like 公式, 定理, 解题关键, 易错, 步骤1, 步骤2',
    '语文': 'Chinese labels like 修辞手法, 中心思想, 关键词, 段落大意, 写作手法',
    '文言': 'classical Chinese labels like 实词, 虚词, 古今异义, 词类活用, 特殊句式',
    '物理': 'physics labels like 公式, 原理, 受力分析, 解题关键, 易错点',
    '化学': 'chemistry labels like 反应方程式, 实验步骤, 注意安全, 易错',
    '通用': 'study labels like 重点, 难点, 考点, 易错, 必背, 理解',
}
```

调用时传入 subject 参数：
```python
apply_aging_and_annotations(img, analysis, output, subject='数学')
```

## 批注内容示例

### 英语书页面
- 红色批注："必考！"
- 蓝色边栏："主语"、"谓语"、"宾语"
- 红色圆圈：that, which, where 等关系代词

### 数学书页面
- 红色批注："核心定义"
- 蓝色边栏："公式"、"定理"、"解题关键"
- 红色圆圈：积分符号、微分符号

### 文言文书页面
- 红色批注："必背！"
- 蓝色边栏："实词"、"虚词"、"古今异义"
- 红色圆圈：重点文言词汇
